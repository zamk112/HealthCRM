# HealthCRM

A secure, production-ready CRM template built for environments with stringent data handling requirements. This project serves as a portfolio showcase of modern full-stack architecture, with a focus on following best practices in all facets of building reliable secure web applications. 

## Overview

Built as a portfolio project demonstrating modern full-stack architecture with:

- **ReactJS** (TypeScript) — component-driven frontend with type safety
- **ASP.NET Core Web API** — RESTful backend with clean architecture
- **Docker** — fully containerised development and deployment
- **NGINX** — reverse proxy with SSL termination
- **OIDC / OAuth2** — OpenID Connect authentication with role-based access control
- **PDF Generation** — server-side document rendering
